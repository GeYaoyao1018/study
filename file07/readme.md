# 聞き取り練習1
来月の消費税率の引き上げて、食品や飲み物や店内で飲食の場合は10パーセント　持ち帰りのばあいは軽減税率の8％を適用され
この二つの税率にどう適用するか　外食各社の価格戦略が注目されています。
牛丼チィーンの最大手のは主力商品の牛丼並盛りを店内飲食する場合の本体価格を値下げし税込価格では持ちかえた場合と同じ方針を固みました。数奇屋の牛丼なみむり本体価格は現在325円ですが
店内飲食場合六円下げして、319円にします　10％の消費税を加えた税込価格今と同じ350円に据え置き　持ち帰りの税込価格と同じにします。
会社としては消費者の節約志向は根強いと見てみて主力商品の実質的値下げでお客と繋ぎ狙いがあります。他の牛丼チィエン-では
吉野家が本体価格をすいおういて店内飲食と持ち帰りの税込価格を別にします。店内飲食と持ち帰りそれぞれの場合、消費者が実際に支払う税込の
価格を同じにするか別別にするか来月の一日消費税率の引き上げに向けって、外食各社の対応は分かれています。
消費者の根強い節約志向に続く中、商品の価格設定は売上を大きく左右するだけに各社にとっては重要な経営判断です。ファストフードではケンタっき
フライドチキンは主力商品を店内で飲食する場合も本体価格を値下げしい、税込価格今と同じにすいおうきます。それによって、店内飲食と持ち帰り
の税込価格を統一します。
一方、モスバーガーは全ての商品について本体価格は今と同じすいおうします。店内飲食と持ち帰りで税込価格に差を付ける方針です。商品税率
引き上げのギリギリまで対応検討する会社もあり、価格設定しだいでは客離れを引き起こしかねないという危機感の現れとも言えます。

# 聞き取り練習２
若い世代保険への加入を増やそうとスマートフォンを使った、保険料がいちにちにあたり数百円の小額保険が相次いで登場しています。
生命保険の大手の第一生命はスポーズ大会が祭りなどの怪我や食中毒に備えて一日あたり百元前後の保険料で最大五万円の入院費を補償する商品をはつばいしました。
スマートフォンのアプリで契約や保険料の支払い、それに怪我の診断書などをカメラで撮影して登録すれば保険金を請求もできます。
また損害保険大手の。。。。会社　去年から通信アプリの大手のラインとともに車を借りた際の自動車保険は旅行保険などの小額保険を販売しています。
各社によりますと、三十代以下の保険加入率は四十代、五十代くれべて低いということで小額保険で若い世代を取り組みたい考えています。
